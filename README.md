# FUTURE_DS_01
Business Sales Performance Analytics (Superstore Dataset)

This project focuses on analyzing and visualizing business sales performance using the Superstore Sales Dataset. The main objective is to extract meaningful insights that help in understanding sales trends, customer behavior, and overall business performance.

As part of my internship at Future Intern, I worked on end-to-end data analysis, including data cleaning, processing, and dashboard creation.

🔧 Tools & Technologies Used:
Python (Pandas) – for data cleaning and preprocessing
Microsoft Excel – for initial data exploration and formatting
Power BI – for interactive dashboard creation and data visualization

📌 Key Responsibilities:
Cleaned and transformed raw dataset using Pandas (handled missing values, duplicates, and formatting issues)
Performed data validation and preprocessing in Excel
Built interactive Power BI dashboards to analyze:
Sales performance
Profit trends
Customer segments
Regional performance

📈 Key Insights:
Identified high-performing regions and products
Analyzed the impact of discounts on profit
Tracked sales trends over time
Discovered customer purchasing patterns
